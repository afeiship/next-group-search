# webkit-sassui-icon-badge
> webkit sass ui for icon with badge.

## usage:
```scss
@import 'bower_compents/webkit-sassui-icon-badge/dist/webkit-sassui-icon-badge.scss';
@include webkit-sassui-icon-badge(
  $inSize:48px,
  $inRadius:10px,
  $inBadgeColor:#FFF,
  $inBadgeBgColor:#dd514c,
  $inBadgeRadius:10px
)
```
